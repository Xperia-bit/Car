# STM32 多功能智能小车

一个基于 STM32F103C8T6 的多功能智能小车项目，支持蓝牙遥控、自动避障、红外循迹和 OLED 状态显示。

本项目最初是为了解决传统面包板智能小车接线杂乱、容易松动、调试不方便的问题，将常用的小车模块整理到一块 PCB 上，方便焊接、复现、调试和二次开发。

## 项目简介

本项目使用 STM32F103C8T6 作为主控，TB6612 作为电机驱动芯片，可以控制四轮小车底盘运动。同时板载或预留了常见智能车模块接口，包括：

- TB6612 电机驱动
- 蓝牙串口模块
- HC-SR04 超声波测距模块
- SG90 舵机
- 五路红外循迹模块
- 0.96 寸 IIC OLED 屏幕
- 12V 电源输入与 5V 供电接口

程序有 HAL 库和标准库，代码结构比较简单，适合 STM32 入门、小车课程设计、电子实训和智能车功能验证。

## 功能特点

- 支持蓝牙串口遥控
- 支持自动避障
- 支持五路红外循迹
- 支持 OLED 显示当前模式和传感器状态
- 使用 TB6612 驱动左右两组直流电机
- 使用舵机带动超声波模块左右扫描
- PCB 集成常用接口，减少杜邦线和面包板连接
- 代码按模块拆分，方便阅读和修改

## 功能模式

程序中目前包含三种工作模式：

| 模式编号 | 模式名称 | 说明 |
| --- | --- | --- |
| 0 | 蓝牙遥控模式 | 通过手机蓝牙串口助手发送指令控制小车运动 |
| 1 | 自动避障模式 | 通过超声波测距和舵机扫描实现简单避障 |
| 2 | 红外循迹模式 | 通过五路红外模块检测黑线并自动调整方向 |

程序默认启动后进入循迹模式。

## 蓝牙控制指令

蓝牙模块连接 STM32 的 USART1，手机端可以使用任意蓝牙串口助手发送单字符指令。

| 指令 | 功能 |
| --- | --- |
| `0` | 前进 |
| `1` | 后退 |
| `2` | 左转 |
| `3` | 右转 |
| `4` | 原地左转 |
| `5` | 原地右转 |
| `6` | 刹车 |
| `7` | 切换到蓝牙遥控模式 |
| `8` | 切换到自动避障模式 |
| `9` | 切换到红外循迹模式 |

注意：`0` 到 `6` 的运动控制指令只有在蓝牙遥控模式下才会执行。

## 硬件组成

| 模块 | 型号 / 说明 | 作用 |
| --- | --- | --- |
| 主控 | STM32F103C8T6 | 负责小车整体控制 |
| 电机驱动 | TB6612 | 控制左右两组电机正反转和速度 |
| 蓝牙模块 | HC-05 / HC-06 等串口蓝牙模块 | 手机遥控和模式切换 |
| 超声波模块 | HC-SR04 | 测量前方障碍物距离 |
| 舵机 | SG90 或兼容舵机 | 带动超声波模块左右扫描 |
| OLED | 0.96 寸 IIC OLED | 显示模式、距离、传感器状态 |
| 循迹模块 | 五路红外循迹模块 | 检测黑线位置 |
| 电源 | 12V 输入，5V 输出 | 给电机和模块供电 |

## 主要引脚连接

### 电机驱动 TB6612

| 功能 | STM32 引脚 |
| --- | --- |
| AIN1 | PB15 |
| AIN2 | PB14 |
| BIN1 | PB13 |
| BIN2 | PB12 |
| PWMA | PA8 / TIM1_CH1 |
| PWMB | PA11 / TIM1_CH4 |

### 蓝牙串口

| 功能 | STM32 引脚 |
| --- | --- |
| USART1_TX | PA9 |
| USART1_RX | PA10 |

蓝牙模块接线时需要注意 TX/RX 交叉连接：

- 蓝牙 TX 接 STM32 RX
- 蓝牙 RX 接 STM32 TX

### 超声波模块

| 功能 | STM32 引脚 |
| --- | --- |
| TRIG | PA0 |
| ECHO | PA1 |

### 五路红外循迹模块

| 功能 | STM32 引脚 |
| --- | --- |
| OUT1 | PA3 |
| OUT2 | PA4 |
| OUT3 | PA5 |
| OUT4 | PA6 |
| OUT5 | PA7 |

代码中默认：

- `1` 表示检测到白色区域
- `0` 表示检测到黑线

### OLED 屏幕

| 功能 | STM32 引脚 |
| --- | --- |
| SCL | PB8 |
| SDA | PB9 |

OLED 使用软件 IIC 驱动。

### 舵机

| 功能 | STM32 引脚 |
| --- | --- |
| Servo PWM | PB0 / TIM3_CH3 |
